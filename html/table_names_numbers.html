WHAT ARE TABLE NAMES / NUMBERS

Table names or numbers sit on each table at your reception. As your guests enter the reception they will view the table plan or pick up an escort card to find out which table they are sitting on.
They simply view the table names / numbers to find the correct table.

The table names are designed to be placed in holders. Venues usually provide these for wedding receptions, but it is always worth checking just in case.

Our table names measure 20cm x 15cm as standard, but we can make them any size for you.
We use lovely thick boards to make your table names so they are guaranteed not to flop!
